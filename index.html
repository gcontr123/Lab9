lab

For this lab, you'll create a Material Components Single Page App that has 3 screens (3 different sets of information or different displays of the same information.) Requirements: App uses Material Components for the Web Includes a nav bar Includes a drawer with navigation Clicking an item in the drawer loads a new page and closes the drawer Submission: Your page should be viewable at and you should submit in Gradescope. Resources: https://material-components.github.io/material-components-web-catalog/#/ https://jquery.com/ (event handlers, show/hide, load) https://en.wikipedia.org/wiki/Single-page_application
